# 🚀 Algoritmos y Estructuras de Datos - Ejercicios

¡Bienvenidos al repositorio de soporte para la materia! Este espacio está pensado para centralizar los códigos de ejemplo, ejercicios resueltos en clase y material complementario en **C++**. 

El objetivo es que tengan un punto de referencia para repasar las guías prácticas y afianzar los conceptos teóricos de la cursada.

---

## 🛠️ Requisitos Previos e Instalación

Para poder correr estos ejercicios en sus computadoras, van a necesitar un compilador de C++ configurado.

### 1. Compilador recomendado
* **Windows:** [MinGW-w64](https://www.mingw-w64.org/) (incluido típicamente con Code::Blocks o configurado para VS Code).
* **Linux/MacOS:** `gcc` / `g++` (disponible nativamente o vía paquetes esenciales).
